# UNP_SQL_PROJECT_BANKING_DATASET_ANALYSIS

 Description:
This project uses SQL to analyze banking data like customers, accounts, and transactions. It helps understand how customers use banking services and find useful information such as high-value customers and transaction patterns.
The project also helps in improving bank decisions, customer service, and detecting unusual activities.
